# Metrics

## Introduction
Make data-driven decisions at each step of your customer journey with Metrics. Easily visualize trends, spot issues and track your CX progress within your journey maps.

When you create a Metric, you will need to link it to a data source. Metrics can be created via an integration (coming soon) or added manually.
